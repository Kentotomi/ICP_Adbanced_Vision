# ICP_Adbanced_Vision

2022年度のアドバンストビジョンの課題2でICPアルゴリズムを実装した．

## 必要なツールやライブラリのインストール

```bash
# Python3をインストールする
$ sudo apt-get install python3

# numpyをインストールする
$ pip3 install numpy

# open3dをインストールする
$ pip3 install open3d
```

## 実行手順

```bash
$ git clone https://github.com/Kentotomi/ICP_Adbanced_Vision.git

$ cd src

$ python icp.py
